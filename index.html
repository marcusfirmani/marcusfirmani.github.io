Projects

“MapMission” iOS App:

Map exploration and learning app that uses gamification to improve the user’s knowledge of geography. App store link: http://www.appstore.com/mapmission

  • Built custom overlay to MapKit to enable accurate swipe counting for game use.
  • Used GameKit to allow scorekeeping, friend challenges and leaderboards.
  • Produced graphics and icons in Gimp and game sounds in Garageband.

“Grover” Android App:

Math collaboration app built for nonprofit Philadelphia Area Math Teachers’ Circle (group project for Mobile Development course). App store link: https://goo.gl/mTvb6R

  • Created post feeds using RecyclerView and CardView.
  • Enabled taking and uploading of photos to remote server.
  • Designed database schema and implemented it with the Parse BaaS.

Registrar Scheduling Program in Java:

Determines optimal schedule of classes based on student preferences and other constraints (group project for Algorithms course).

  • Parsed text input files and produced output file with generated schedule.

HERA Microprocessor:

CPU simulation built in Logisim that runs HERA assembly language code.

  • Used logic gates and wires and created modules.

Image Binarization Program in Python:

Enables the user to apply various binarization filters to an image using buttons and see the resulting image preview before exporting the final image.

  • Created GUI with Tkinter.
  • Worked with raw image data using PIL (Python Imaging Library).

Tiger Compiler:

Built for Compiler Design course, spring 2016.
